🔧 Como Configurar a Automação no Google Apps Script
  1. Abra o Google Sheets com a planilha RELAÇÃO GERAL COLABORADORES CGH
  2. Acesse Extensões → Apps Script
  3. Cole o conteúdo do arquivo apps-script/automacao.gs (incluso neste projeto)
  4. Ajuste os e-mails dos administradores na função enviarResumoAdmin()
  5. Execute testarEnvioEmail() para validar o template
  6. Crie o gatilho: Adicionar Gatilho → verificarEEnviarAlertas → Diário → 08:00
  7. Autorize as permissões quando solicitado
